Tic Tac Toe using C++

In a Tic Tac Toe game implemented in C++, a 3x3 grid is represented using a 2D array initialized with empty spaces. Players take turns entering row and column coordinates to place their marks ('X' or 'O') on the board. After each move, the board is displayed, illustrating the current game state. A win condition is checked by examining rows, columns, and diagonals for three consecutive marks. The game continues until a player wins or the board fills up, resulting in a draw. The game flow alternates between players, ensuring they make valid moves and tracking the progress until a conclusive outcome is reached, delivering an engaging and interactive gameplay experience.
